萌声・ロリ声のF0（基本周波数）分析

一般的な声のF0範囲

カテゴリ F0範囲 (Hz)
成人男性 85-180 Hz
成人女性 165-255 Hz
子供（女児） 250-400 Hz
ロリ声 300-500+ Hz
萌声 250-400 Hz
ロリ声の特徴

F0: 350-500 Hz（平均約400 Hz）

基本周波数が高い - 子供の声域を模倣
フォルマント周波数も高い - 声道が短い子供の特徴を再現
ピッチ変動が大きい - 抑揚が激しい（5-10 Hzの細かい揺れ）
呼吸的発声 - 息成分が多く、H1-H2差が大きい
萌声の特徴

F0: 280-380 Hz（平均約320 Hz）

ロリ声よりやや低め - 成人女性の可愛い声に近い
ソフト発声 - 気息成分多め、声帯閉鎖が弱め
ピッチ変動パターン - 語尾上げが多い（↑5-15 Hz）
ハスキー成分 - わずかなノイズ成分が「可愛さ」を演出
分析研究のポイント

測定方法：

Praatなどの音声分析ソフト
F0抽出アルゴリズム（autocorrelation法）
フォルマント分析（F1, F2, F3）
キャラクター別の例（概算）：

キャラタイプ F0平均 特徴
ロリ系 380-450 Hz 高F0、高F1、強いピッチ揺れ
ツンデレ 300-350 Hz 通常域→怒り時は350+ Hz
お姉さん系 220-280 Hz 低F0、安定ピッチ
マスコット系 400-600 Hz 超高域、非現実的
声優が使うテクニック

頭声（地声より高い位置） - F0を物理的に上げる
声道短縮イメージ - フォルマントを高くする
リラックスした喉 - 息成分を増やす
舌の位置調整 - 前に持ってくることで声道を短くする
注意点

個人差が大きい
録音環境・マイク特性でも変わる
地声のF0と「演じる声」のF0は異なる
長時間のロリ声は声帯への負担が大きい
